# Mesa Landing Page

A beautiful, modern landing page for Mesa - an on-demand student help network.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Modern UI**: Clean, rounded elements with soft sky-blue and violet accents
- **Smooth Scrolling**: Interactive navigation between sections
- **Accessible**: Semantic HTML and proper contrast ratios

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS 3
- TypeScript
